#CD計算機
說明:簡單計算出CD的時間

#主砲
* 面板裝填為艦船不穿裝備的面板值
![image](https://cdn.discordapp.com/attachments/624599897798213632/720270465238827068/MuMu20200610213113.png)  
* 裝填增加BUFF 傻白、長門..等。填入百分比之前的數值15%則填入15  
如有多個則加算，如15%與20%則填入35。
* 裝備、貓、科技增加裝填值加算後填入即可  
* 如有裝備金火控則將金火控開關打開
* 最後按計算按鈕即可算出CD
![image](https://cdn.discordapp.com/attachments/624599897798213632/720276196298063922/2020-06-10_22-00-13.jpg)

#空襲CD
* 先選擇空母  
![image](https://cdn.discordapp.com/attachments/624599897798213632/720277373706305626/2020-06-10_22-04-59.jpg)
* 時間縮短 如赤城、加賀有先手必勝技能50%則填入50
* 選擇艦載機 由上至下為艦載機1、艦載機2、艦載機3。  
選擇裝備的艦載機即可算出時間

#可畏計算機
* 預設第三格為青花魚
* 計算後得出各式裝備組合
